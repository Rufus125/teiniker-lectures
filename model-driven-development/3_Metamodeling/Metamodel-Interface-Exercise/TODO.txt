Exercise: Interface Metamodel 
---------------------------------------------------------------------

1. Entwerfen Sie ein Metamodel, welches die Modellierung einfacher
	Java Interfaces ermöglicht.
	
	Beispiel: 
	
		public interface Stack
		{
			void push(int value);
		
			int pop();
		
			int top();
		
			boolean isEmpty();
		
			boolean isFull();
		}
		
		
2. Implementieren Sie die nötigen Metaklassen und überprüfen Sie Ihre
	Implementierung mit der Klasse "InterfaceMetamodelTest".
